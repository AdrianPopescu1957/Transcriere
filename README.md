# Transcriere — Generare minute ședință

Director proiect: D:/Proiecte/IA/Transcriere

Scop:
- Proiect pentru generarea minutelor unei ședințe pornind de la o transcriere în limba română.

Cum se testează (Windows):
- Am adăugat scriptul `run_test.bat` în rădăcina proiectului. Acesta schimbă directorul la calea proiectului și încearcă să ruleze `main.py` sau `src\main.py` cu Python.
- Dacă aplicația ta are un alt punct de intrare, modifică `run_test.bat` sau rulează manual comanda dorită (de exemplu `python -m src.main` sau `python app.py`).

Instrucțiuni importante:
- Nu inventa informații care nu apar în transcriere.
- Fii concis, tehnic și neutru.
- La final, oferă un sumar de maximum 5 rânduri.

Fișiere adăugate:
- `prompt_minute.txt` (conține instrucțiunile de generare minute)
- `run_test.bat` (script utilitar pentru testare pe Windows)
